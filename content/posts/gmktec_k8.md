It comes with AMD Radeon 780M Gaming PC GPU.
It is an integrated GPU based on the new RDNA3 architecture and has 12 CUs (= 768 shaders) clocked at up to 2700MHz. 

It contains an Oculink Interface for High-Speed Connectivity by allows ultra-fast external connections for storage expansion or GPU upgrades. 
This ensures faster data transfer rates compared to traditional interfaces, offering enhanced flexibility for tasks like large data transfers or external GPU use.

The Nucbox K8 Plus mini PC supports up to 4 displays in 4K/8K resolution, via 2 USB4 ports (40Gbps), HDMI 2.1 (8K@60Hz), and DisplayPort 2.1 (8K@60Hz). 
The USB4 ports support Power Delivery 3.0 and DP1.4 for seamless multitasking, allowing users to run multiple applications simultaneously.

It also featuring three customizable performance modes: Silent mode (35W), Balanced mode (54W), and Performance mode (65W, peak 70W). 
This flexibility ensures that you can optimize for either silent operation or maximum performance depending on your workload.
