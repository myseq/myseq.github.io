## Multicloud Cmdline

Brandon Evans (from SANS) has created a multicloud command line interface cheat sheet.
It helps to perform common cloud tasks including connecting to the cloud, filtering, querying data, SSH to virtual machine and managing storage.

The cheat sheet covers different commands for AWS, Azure and GCP.
The demo below includes how to list S3 buckets, download/upload files, and perform encryption/decryption using AWS KMS.

See the links below to download this handy cheat sheet.
